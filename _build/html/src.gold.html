src.gold.sending_gold module

src.gold.sending_gold.gold_table(dataframe: DataFrame) → None[source]

Inserta los datos contenidos en el DataFrame proporcionado en la tabla ‘ventas_gold’ dentro de la base de datos SQLite ‘coppelchallenge.db’.

La función crea la tabla si no existe, con las columnas: Quantity, total, Target_90Days y regroup_country.

Args:
dataframe (pd.DataFrame): DataFrame que contiene los datos a insertar.

Debe incluir las columnas: ‘Quantity’, ‘total’, ‘Target_90Days’, ‘regroup_country’.

Returns:

None

Raises:

sqlite3.Error: Si ocurre algún error al conectar o ejecutar la consulta SQL.
