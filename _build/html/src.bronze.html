src.bronze.bronze_data module

src.bronze.bronze_data.bronze_table() → None[source]

Carga un archivo CSV con datos de transacciones retail y lo inserta en una tabla SQLite llamada ‘ventas_bronze’ dentro de la base de datos ‘coppelchallenge.db’.

La función crea la tabla si no existe, con las columnas especificadas.

No recibe parámetros ni retorna valor.

Raises:

pd.errors.ParserError: Si hay problemas leyendo el archivo CSV. sqlite3.Error: Si hay errores en la conexión o ejecución SQL.
