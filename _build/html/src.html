src.app.receive_data(data: List[Data]) → Dict[str, Any][source]

Recibe una lista de objetos Data, los procesa y genera predicciones.

Flujo: - Convierte la lista en DataFrame. - Inserta datos en tabla ‘ventas_bronze’. - Aplica transformaciones Silver y guarda en ‘ventas_silver’. - Aplica transformaciones Gold. - Prepara DataFrame para predicción. - Realiza predicción con el modelo cargado. - Guarda resultados en ‘outputs_table’. - Retorna el DataFrame resultante en formato JSON.

Args:

data (List[Data]): Lista de objetos con la información de ventas.

Returns:

Dict[str, Any]: Diccionario con clave “data” y valor lista de registros con predicciones.
